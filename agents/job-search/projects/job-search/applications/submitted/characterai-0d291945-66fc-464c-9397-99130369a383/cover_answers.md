## [Optional] What is one feature you would be excited to add or improve?

I would focus on improving the onboarding moment where a new user first picks or discovers a character. Right now that choice happens early, under low context, and it is high-stakes for activation - if someone picks a character that does not resonate in the first session, they are unlikely to come back and try a different one. I would want to run experiments around a lightweight "character matching" flow: a few quick preference signals during onboarding (tone, topic, vibe) that surface a curated shortlist rather than dropping users into a cold browse. The goal is not a quiz for its own sake but a faster path to the first emotionally resonant conversation, which I would expect to show up clearly in D1 retention.

The instrumentation question is just as interesting as the feature itself. You would want to track not just whether users picked a character but whether that first conversation crossed some engagement threshold - message depth, session length, something that proxies for "aha." That signal becomes the optimization target for every variant. It is the kind of experiment where getting the metric definition right is half the work, and that is the part I find most interesting to own.
