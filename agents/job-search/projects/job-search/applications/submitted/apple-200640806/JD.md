## Summary

We’re looking for a firmware engineer to contribute to the next Wi-Fi products that Apple is building. The candidate will design, develop and test firmware code implementing different functionality of Apple's Wi-Fi solution.

## Description

The candidate will work with hardware and systems engineers developing requirements for multiple features, designing HW/SW split and working with test engineers to integrate and test their features.

## Responsibilities

You will develop firmware code for Apple’s Wi-Fi chip sets
You will design / code / integrate code in C/C++ for Wi-Fi based algorithms to implement the 802.11 protocol
You will work with test teams to validate the developed features
Follow process for releasing features

## Minimum Qualifications

You’re fluent in C/C++
You understand wireless communications and have networking stack exposure
You understand communication theory and can contribute to technical discussions regarding wireless systems
You’re fluent in Git or other code source management tools
You thrive in a collaborative environment and can clearly communicate while driving multiple projects across teams
You consistently achieve results, even under tough circumstances
You possess a BS in electrical engineering or equivalent

## Preferred Qualifications

You have a MS in electrical engineering or equivalent
You have 5+ years experience
You enjoy taking on new opportunities and tough challenges with a sense of urgency, high energy, and enthusiasm
You build partnerships and work collaboratively with others to meet shared objectives
