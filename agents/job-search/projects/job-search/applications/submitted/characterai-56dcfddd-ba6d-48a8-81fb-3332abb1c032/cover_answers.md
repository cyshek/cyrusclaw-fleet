## [Optional] What is one feature you would be excited to add or improve?

I would focus on a persistent memory layer that actually works at scale across long-term user sessions. Right now, a lot of AI character interactions feel episodic - each conversation starts relatively fresh, and the character loses the accumulated context that would make ongoing relationships feel real and meaningful. Users who come back daily to the same character are essentially re-introducing themselves over time, which limits the depth of connection the platform could otherwise create.

The product challenge is not just technical - it is also about giving users transparency and control over what the character remembers, so the experience feels trustworthy rather than opaque. I would want to design a lightweight memory management surface where users can see key things the character has learned about them, adjust or remove entries, and understand how memory shapes responses. That kind of user agency would deepen engagement and trust simultaneously.

From a TPM lens, this is the type of feature that touches model infrastructure, product design, trust and safety, and privacy policy all at once - exactly the cross-functional coordination problem I find most energizing. I built coordination processes at Microsoft to keep multi-team initiatives from stalling, and a feature like this would demand that same discipline to ship well.
