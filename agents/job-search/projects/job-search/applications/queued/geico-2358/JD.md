# Technical Program Manager - Data Center Hardware Implementation @ GEICO (R0061267)

GEICO is seeking a Technical Program Manager to oversee Data Center Environment (DCE) Implementation, managing project plans, logistics, and ensuring timely delivery within budget.

## Responsibilities
- Develop and maintain project plans, including timelines and milestones.
- Create and manage ordering schedules for equipment and materials.
- Coordinate logistics to ensure smooth delivery and installation.
- Oversee hardware evaluation schedules and associated activities.
- Identify and manage provisioning dependencies; oversee exit-related activities and transitions.
- Oversee implementation of projects and systems; ensure on-time, on-budget delivery.
- Communicate project status, risks, and updates to stakeholders; maintain comprehensive project documentation.
- Manage power requirements, connectivity setups, and activity-based colocation/project billing reviews.
- Work with sourcing and supplier management teams to ensure contracts, orders, and parts arrive on time.
- Work with capacity planning teams to track current utilization vs anticipated demand.

## Qualifications
- Data center technologies, server hardware NPI, component vendor supply chain, networking, servers, GPUs, storage, virtualized infrastructure, power management, cooling systems.
- Project planning, logistics coordination, provisioning management.
- Clear communication of complex requirements (schedules, orders/shipment tracking, system configs) across teams.
