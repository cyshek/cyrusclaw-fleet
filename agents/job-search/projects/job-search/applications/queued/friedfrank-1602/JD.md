Position Summary:

The Cloud Solution Architect is responsible for defining the enterprise cloud strategy, architecting scalable, secure cloud-native platforms, and leading the implementation of AI-driven solutions across IaaS, PaaS, and SaaS environments. This role provides technical leadership for cloud transformation initiatives leveraging Microsoft Azure, AWS, AI/ML services, Databricks, and modern DevOps methodologies.

The Cloud Solution Architect drives enterprise cloud modernization programs, establishes governance standards, and leads cross-functional teams in designing highly available, resilient, and cost-optimized cloud architectures. This role requires deep expertise in hybrid cloud infrastructure, enterprise migrations, cloud security, Infrastructure as Code (IaC), and cloud operational excellence.

The ideal candidate combines strong hands-on technical expertise with strategic leadership to align business objectives with enterprise technology roadmaps.

Duties & Responsibilities:

Lead enterprise cloud architecture strategy and roadmap across AWS, Azure, and/or Google Cloud Platform (GCP)
Design scalable, secure, highly available cloud-native architectures for business-critical applications
Drive cloud transformation and modernization initiatives, including migration of on-premises systems to cloud environments
Architect AI-driven and data-centric solutions using services such as Machine Learning, Cognitive Services, Databricks, and Generative AI platforms
Establish cloud governance frameworks, security standards, operational best practices, and compliance controls
Define and implement Infrastructure-as-Code (IaC) solutions using Terraform, ARM/Bicep, CloudFormation, or similar tools
Lead DevOps and CI/CD implementation strategies using modern automation tools and practices
Collaborate with security teams to ensure cloud environments meet enterprise security and regulatory requirements
Optimize cloud cost management, resource utilization, scalability, and performance
Provide architectural guidance and technical leadership to engineering and operations teams
Evaluate emerging cloud technologies and recommend strategic adoption opportunities
Develop reference architectures, technical standards, and cloud operating models
Support disaster recovery, business continuity, and high-availability planning
Work closely with executive leadership and stakeholders to align cloud strategy with business objectives
Mentor cloud engineers, solution architects, and technical teams

Preferred Qualifications: 

Bachelor’s degree in computer science or Cloud Computing, or alternatively, 5-8 years of pertinent experience in cloud architecture and solution design
Cloud certifications such as:
AWS Certification Solutions Architect – Professional
Microsoft Certification: Azure Solutions Architect Expert
Google Professional Cloud Architect
Experience with enterprise architecture frameworks
Familiarity with FinOps and cloud cost optimization practices
Experience leading large-scale digital transformation programs

Technical Skills:
