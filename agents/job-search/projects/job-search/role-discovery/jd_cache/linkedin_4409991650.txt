# Solutions Architect

Position: Solutions Architect
Compensation: $70 - $100 /hour, depending on experience
Location: Oklahoma City, Oklahoma
Inceed has partnered with a great company to help find a skilled Solutions Architect to join their team!
This is an exciting opportunity to work with a dynamic team focused on enhancing database and server performance. The role is critical to resolving CPU saturation issues during peak loads. Join a forward-thinking company that values technology advancements and internal growth. This contract position is open due to the need for expert guidance in optimizing server configurations.
Key Responsibilities & Duties:
Review database and server architecture for improvements
Conduct assessments of server configurations
Provide recommendations to prevent CPU saturation
Analyze resource utilization and suggest best practices
Access and evaluate all systems for issue identification
Perform load testing and network analysis
Update and modify network settings, stored procedures, and queries
Required Qualifications & Experience:
Expertise in SQL and database management
Experience with server and network configuration
Strong analytical and problem-solving skills
Ability to conduct thorough system assessments
Proficiency in identifying and resolving performance issues
Nice to Have Skills & Experience:
Background in application development
Experience with load testing tools
Knowledge of best practices in server architecture
Perks & Benefits:

Weekly direct deposit or pay card deposit
