You Are
We are seeking a visionary Technical Solution Architect to lead the modernization of a large-scale retail enterprise, transitioning from legacy AS/400 systems to cloud-native platforms on Google Cloud Platform (GCP) or Microsoft Azure.
The Opportunity
Architect and lead the re-platforming of legacy AS/400 applications to cloud-native solutions on GCP or Azure.
Reimagine and redesign enterprise applications for scalability, performance, and modern user experiences.
Define and implement integration strategies across internal systems, third-party platforms, and cloud services.
Lead AI/ML-based development initiatives, enabling intelligent automation and predictive analytics.
Design and oversee data warehouse and data analytics solutions to support high-volume OLTP systems and varied user interactions.
Drive backend architecture using microservices, RESTful APIs, and event-driven systems.
Guide frontend modernization using frameworks like React.js, Next.js, and TypeScript, ensuring responsive and accessible UI/UX.
Oversee DevOps practices, including CI/CD pipelines, Infrastructure as Code (IaC), container orchestration, and cloud-native deployment.
This position description identifies the responsibilities and tasks typically associated with the performance of the position. Other relevant essential functions may be required.
What You Need
Experience Level: 10+ Years
10+ years of experience in enterprise architecture, solution design, and cloud transformation.
Proven experience migrating from AS/400 or other legacy systems to GCP or Azure.
Deep understanding of microservices, event-driven architecture, and API-first design.
Strong background in AI/ML platforms, model lifecycle management, and data engineering.
Expertise in data warehousing, ETL pipelines, and real-time analytics.
Experience with high-throughput OLTP systems and designing for performance and reliability.
Hands-on experience with CI/CD, DevOps, and Infrastructure as Code (IaC).
Solid experience with backend technologies (Java, Spring Boot, Node.js, Python).
Strong proficiency in frontend frameworks (React.js, Next.js, TypeScript, HTML/CSS).
Familiarity with containerization (Docker, Kubernetes) and cloud-native deployment.
Experience with security frameworks, IAM, and compliance standards.
Excellent communication and stakeholder management skills in client-facing roles.
Experience working in Agile/Scrum environments and leading onsite-offshore teams.
Preferred Tools & Technologies:
Cloud Platforms: GCP (BigQuery, Cloud Functions, Pub/Sub), Azure (Synapse, Logic Apps, AKS)
Backend: Java, Spring Boot, Node.js, Python
Frontend: React.js, Next.js, TypeScript, HTML5, CSS3, Tailwind
AI/ML: Vertex AI, Azure ML, TensorFlow, PyTorch
Data & Analytics: BigQuery, Azure Synapse, Snowflake, Kafka
DevOps & IaC: Terraform, GitHub Actions, Jenkins, Helm, Docker, Kubernetes
Monitoring & Security: Stackdriver, Azure Monitor, ELK, IAM, OAuth2
Collaboration: Jira, Confluence, Slack, Microsoft Teams
