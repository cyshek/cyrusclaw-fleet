# Forward Deployed Engineer

Company Overview
Domo's AI and Data Products Platform lets people channel AI and data into innovative uses that deliver a measurable impact. Anyone can use Domo to prepare, analyze, visualize, automate, and build data products that are amplified by AI.
Position Summary
We are seeking a highly technical, customer-facing Forward Deployed Engineer to help redefine how enterprise software is built and delivered. In this role, you will design and deploy enterprise-grade applications and intelligent agent workflows directly on Domo’s platform—replacing traditional SaaS products with custom-built, real-time solutions.
This is not a traditional engineering or consulting role. You will work side-by-side with customers, building live solutions using AI-assisted development tools, demonstrating value in real time, and owning the full lifecycle from discovery through deployment and impact measurement. Success in this role requires a unique blend of software engineering expertise, customer engagement skills, and the ability to leverage AI to dramatically accelerate development.
Key Responsibilities
Build and deploy enterprise-grade applications using Domo’s Pro Code Editor, React, and JavaScript;
Lead end-to-end customer engagements, including discovery, solution design, live demos, sprint delivery, and impact measurement;
Develop and implement agentic workflows, including orchestration agents, automations, and human-in-the-loop systems;
Leverage AI-assisted development tools (e.g., Claude Code, GitHub Copilot) to accelerate build speed and enhance solution quality;
Conduct live builds and demonstrations for customers, including executive stakeholders and conference audiences;
Translate complex business problems into scalable, data-driven application solutions;
Collaborate with customers to replace legacy SaaS tools with custom-built applications on the Domo platform;
Contribute to internal best practices by developing reusable app patterns, agent templates, and delivery frameworks;
Continuously iterate and ship solutions on a rapid cadence (weekly or faster);
Partner cross-functionally with product, engineering, and go-to-market teams to refine and scale the offering.
Job Requirements
Strong proficiency in React and/or JavaScript with the ability to build polished, interactive web applications;
Experience leveraging AI-assisted development tools as a core part of the development workflow (e.g., Claude Code, GitHub Copilot);
Proven ability to work directly with customers, including running discovery sessions and presenting to technical and non-technical stakeholders;
Solid understanding of enterprise software architecture, including APIs, security, role-based access control, and data governance;
Ability to operate effectively in ambiguous, fast-paced environments with minimal direction;
Strong problem-solving skills with a bias toward action and rapid execution.
Preferred Qualifications
Experience with the Domo platform, including Pro Code Editor, datasets, connectors, and cards;
Hands-on experience building or deploying agentic AI systems (e.g., orchestration agents, autonomous workflows);
Familiarity with agent frameworks such as OpenClaw or similar technologies;
Background in solutions consulting, solutions engineering, or technical account management;
Experience delivering live demos, presentations, or conference talks.
