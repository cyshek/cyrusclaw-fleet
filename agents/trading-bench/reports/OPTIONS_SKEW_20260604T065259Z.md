## STEP 1 — DATA FEASIBILITY (primary finding)

Built on Tessera's prior Alpaca free-tier probe; did NOT re-discover it. New determinations:

### 1a. Options-bar history depth — **SHALLOW, hard floor 2024-01-18**
Every SPY contract's daily-bar history begins **exactly 2024-01-18**, regardless of the contract's listing date (verified: a Dec-2024 LEAPS listed in 2022 still starts 2024-01-18; all 2021/2022/2023-expiry contracts return **NO BARS**). That date is the free-tier OPRA feed floor, not a per-contract artifact.
- **Usable span: 2024-01-18 → present ≈ 16 months.** Clears the ≥1yr bar, but BARELY.
- **Does NOT overlap our 2020-12+ equity history at all.** Zero cross-validation against the regimes (2020 COVID, 2022 bear) where a fear-gauge would matter most.

### 1b. Historical chain enumeration — **BROKEN for the past (had to reconstruct)**
`GET paper-api/v2/options/contracts?underlying_symbols=SPY` returns ONLY currently-listed (unexpired) contracts. Filtering by any **past** `expiration_date` (or `status=all`) returns **0 contracts**. There is **no point-in-time historical-chain endpoint** on this tier.
- **Workaround used:** reconstruct OCC symbols algorithmically from SPY spot (harness `bars_cache`) — pick ATM (round to $5) and ±5% strikes for the nearest standard monthly (3rd-Friday) expiry ≥25 DTE, build `SPY{yymmdd}{C/P}{strike}`, fetch that contract's daily bar. Works, but is a reconstruction, not a true as-of chain snapshot (can miss odd strikes; restricted to $5 grid + monthlies).

### 1c. Liquidity — **adequate near ATM** (better than the n=6 single-sample feared)
Sep-2024 daily contract bars: ATM call ~314 trades/day; OTM put (-5%) ~25–42 trades/day; deep OTM thinner. Enough to get a non-garbage daily close to invert. Thin tails would corrupt 25-delta wing skew, so I used a robust ±5% risk-reversal rather than deep-delta wings.

### 1d. IV must be COMPUTED — confirmed
Bars carry OHLCV only (no greeks/IV). Built a Black-Scholes bisection inverter (r=4.5%, T=DTE/365, SPY spot anchor) over contract closes. Coverage: **596 dates → 551 valid ATM-IV, 550 valid skew (92%)**. IV levels sane (11–23%, spiking on Aug-2024 carry unwind 0.22, Apr-2025 tariff 0.22, Mar-2026 0.23). Leak-safe: signal on date d uses ONLY contract bars with t≤d; harness trades on the d+1 bar (`backtest_xsec` decides pre-bar). No lookahead.

**Feasibility conclusion:** A clean daily skew series IS constructible point-in-time over ~16 months. But the span is short, single-regime (overwhelmingly bull), non-overlapping with our equity history, and chain reconstruction is approximate. **Feasible enough for ONE low-confidence signal test; NOT enough to trust a result.** A proper study needs a paid tier with deep history + true historical chains (Alpaca Algo Trader Plus ~$99/mo for fuller OPRA; or ORATS/CBOE DataShop for clean point-in-time IV surfaces back to ~2007, ~$entry few-hundred/mo). Documented; proceeded to Step 2 because the ≥1yr bar was met.

## STEP 2 — SIGNAL TEST (skew, not IV — see relabel guard)

### RELABEL-vs-DEAD-VOL-LANE check (the gate that picks the signal)
Correlation of each options signal vs SPY trailing-20d realized vol (annualized), n=550:
- **corr(ATM-IV, realized vol) = 0.662** → ATM-IV is a **relabel of the already-rejected vol-level lane (+0.54)**. FLAGGED. **ATM-IV-as-fear-gauge rejected outright — do not pursue; it is the dead lane in disguise.**
- **corr(SKEW, realized vol) = 0.204** → skew is **genuinely distinct** from realized vol. This is the orthogonal, forward-looking crash-pricing input the lane was created to find. **Skew is the only defensible signal here** → tested it.

### Harness-gap note (documented, no protected edit)
`runner.backtest_xsec` processes `buy` (add) and `close` (full exit) only; a `sell` action is treated as a buy-into-existing (cannot partial-reduce). So proportional vol-targeting is impossible through this harness — the signal was implemented as a **binary close/buy SPY timing toggle** on the rolling skew z-score. Protected md5s verified unchanged at finish (runner/backtest/backtest_xsec/risk all match required hashes).

### Results — two panels, $1000 notional, active cost (alpaca_stocks)
Signal: skew z-score (point-in-time, rolling lookback). `fear` mode = high skew→CLOSE (de-risk, the thesis). `contra` = high skew→BUY.

**Buy-&-hold SPY benchmark over the options span:** full-span **FP-cont = +1.350** (SPY +73.9%, near-monotonic bull); overlapping NAMED panel BH FP = +0.901.

**Best skew sweep config:** full-span FP **+1.624** (contra, lb90) — top-8 spread +1.499..+1.624 (looks like a plateau). **But this is a trap — see per-window decomposition:**

| window | regime | BH ret / FP | FEAR-best ret / FP | CONTRA-best ret / FP |
|---|---|---|---|---|
| 2024-Q2 | bull | +22.7% / +2.36 | +22.2% / +2.51 | +20.8% / +2.31 |
| **2025-Q1 tariff** | **bear** | **-1.4% / +0.02** | **-13.0% / -0.97** | +4.6% / +0.39 |
| 2025-Q3 | bull | +9.7% / +0.69 | +0.3% / +0.10 | +16.0% / +1.08 |
| 2026-recent | bull | +14.9% / +1.62 | +15.5% / +2.04 | +22.7% / +2.84 |

### Why this is REJECT, not promote
1. **The actual thesis (FEAR: high skew → de-risk) LOSES.** In the only bear window (2025-Q1 tariff), it returned **-13.0% vs BH -1.4%** (FP -0.97) — it de-risked into the dip and got whipsawed out of the recovery. It also sat in cash through the 2025-Q3 rally (+0.3% vs +9.7%). A fear gauge that hurts you in the one stress window is anti-signal.
2. **The only "winning" variant is contrarian (buy-the-fear), which is NOT a skew edge — it is bull-market beta.** Over 2024-01→2026-06 SPY rose ~+74% with every dip V-recovering; "buy more when skew spikes" mechanically = "buy every dip," which wins *only because this sample never had a dip that didn't bounce*. The contra "win" in the tariff bear (+4.6%) is one lucky V-recovery, not repeatable edge.
3. **The plateau is a mirage.** The tight top-8 spread is the same bull-beta bet at every parameter; it is flat because the sample has essentially one tradable regime. A plateau across a single regime is not robustness.
4. **No ceiling break.** Strip the bull-beta artifact and the thesis-consistent (fear) signal underperforms BH and goes negative-Sharpe in the bear window. Options skew shows **NO sign of carrying genuine timing edge over price history** in the data available — it does NOT break the ~0.5 FP-cont signal ceiling; the >1.3 numbers are bull-beta, identical to BH's own +1.35, not alpha.

## VERDICT: **REJECT** (with FEASIBILITY caveat)
- **Signal:** REJECT. Skew-as-fear-gauge actively hurts in the only stress window; the only positive variant is disguised bull-beta. Zero evidence of orthogonal timing edge. Ceiling NOT broken.
- **Input class:** options-derived skew is genuinely orthogonal to realized vol (corr 0.20, unlike ATM-IV's 0.66 relabel) — so the input class is *real*, but the **free-tier data cannot test it honestly**: 16 months, one regime, no overlap with our history, approximate chains.
- **Honest recommendation:** the next research dollar on this input class should buy **DATA, not more free-tier compute** — a paid IV-surface history (ORATS/CBOE ~back to 2007, or Alpaca Algo Trader Plus) to test skew across multiple bear regimes. Until then this lane is **DATA-LIMITED**, and on free-tier data the answer is REJECT.
