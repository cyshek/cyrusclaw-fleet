## 3. Point-in-time data-honesty justification (the make-or-break of the lane)

We have **only** Alpaca IEX daily bars (~1373 bars from 2020-12, equities + ETFs). We do **not** have a point-in-time macro database. Serving FRED-style final-revised macro series (claims, CPI, PMI, surprise indices) against historical dates is a **silent lookahead leak** — release lag (data for month M prints weeks later) plus revisions (the value you see today is not the value that was knowable on date d). That is an instant disqualifier.

**Therefore the nowcast is built ONLY from TRADEABLE-ETF PRICES, which are point-in-time BY CONSTRUCTION.** A close printed on date d was knowable at date d; ETFs are not revised. Every input is an ETF close sliced to `date <= current SPY bar date`. No revision surface exists, so **no silent lookahead leak is possible.** All proxy symbols fetch clean ~1373-bar history (UUP 1371):

| Sub-signal | Construction (all trailing momentum of ETF closes) | Economic meaning |
|---|---|---|
| growth | mom(XLI)+mom(XLY) − mom(XLP)−mom(XLU) | cyclicals vs defensives |
| credit | mom(HYG) − mom(IEF) | high-yield vs duration (credit risk appetite) |
| curve | mom(SHY) − mom(TLT) | short vs long treasuries (curve/rates) |
| inflation | mom(TIP) − mom(IEF) | breakeven / inflation proxy |
| usd | −mom(UUP) | weak dollar = risk-on |

Each sub-signal is z-scored against its own trailing history (≤120 samples, no lookahead), then the enabled set is averaged into one composite RISK-ON score. This is genuinely exogenous macro structure encoded in prices — **not** a macro database, **not** FRED, **no** release-lag/revision surface.

**Data-blocked note:** a *richer* nowcast (real claims, ISM, surprise indices) would require a paid point-in-time macro vendor (e.g. ALFRED/archival-FRED vintages, or a Bloomberg/Macrobond PIT feed; ~$2k–$24k/yr depending on depth). That is the exact scope of what we'd need to test the "real economic series" version of lane 5 without a leak. We did NOT need it to reach a verdict on the tradeable-proxy version — and the tradeable-proxy version is the honest, leak-free one — so this is a clean REJECT, not a data-blocked memo.

## 4. Relabel diagnostic (the lane's core honesty test)

Correlation of the composite nowcast series with (a) SPY's own 60d trailing return and (b) SPY's 20d trailing realized vol, over a dense 143-point grid:

| Composite config | corr vs SPY return | corr vs SPY vol | interpretation |
|---|---|---|---|
| growth+credit+curve, lb60 | **+0.341** | **−0.434** | moderate; NOT a pure relabel |
| all5, lb60 | +0.294 | −0.346 | moderate |
| **growth ALONE, lb60** | **+0.647** | −0.488 | **a momentum relabel — flagged** |
| credit alone, lb60 | +0.390 | −0.509 | partly vol-ish |
| **curve alone, lb60** | **−0.291** | **−0.026** | **genuinely orthogonal** |
| **growth+credit+curve, lb120** | **−0.106** | **−0.215** | **genuinely orthogonal** |

**Key structural finding:** the `growth` sub-signal alone is largely a SPY-momentum relabel (r=+0.65) — exactly the trap the brief warned about. But the **composite is NOT** a relabel: the `growth+credit+curve` blend sits at r≈+0.34/−0.43, and the longer-lookback (lb120) composite is genuinely orthogonal to both SPY return (−0.106) and SPY vol (−0.215). So the lane's negative result below is an **honest** negative — the signal is real, exogenous, and orthogonal; it just doesn't predict forward returns well enough. We are NOT rejecting a disguised dead lane.

## 6. Plateau vs knife-edge

**This is a knife-edge, not a plateau — a second independent reason to reject.** Within the very same family, the sign flips on lookback:

- spy_binary_zero/gcc: lb120 = **+0.501**, lb60 = −0.079, lb20 = −0.055. The "good" cell is an isolated spike at one lookback, not a coherent ridge.
- xa_tlt vs xa_gld: identical signal, swap the safe asset, and FP goes from −0.40 (TLT) to +0.42 (GLD). The result is dominated by *which safe asset survived 2022's bond crash*, not by the nowcast's timing skill — a tell that the "edge" is an artifact of the safe-leg, not the regime call.
- proportional sizing: uniformly worse and churns 600–1700 trades.

No family shows the soft, all-positive, sign-stable plateau the gate demands.

## 8. Verdict — **REJECT** (honest negative = success)

1. **Best honest FP-cont Sharpe +0.501 < 1.0 gate bar**, AND below the BH-SPY bench (+0.674), AND below every prior rejected lane (vol +0.54, dollar +0.55, dispersion +0.568). The macro nowcast is the WEAKEST signal class tested so far.
2. **Knife-edge, not plateau** — the one good cell is an isolated lookback spike; cross-asset results are dominated by safe-asset survivorship (TLT −0.40 vs GLD +0.42 on the *same* signal), not timing skill.
3. **The relabel trap was checked and largely avoided** — the composite (growth+credit+curve, esp. lb120) is genuinely orthogonal to both SPY return (−0.11) and SPY vol (−0.22). The `growth`-only branch IS a momentum relabel (r=+0.65) and was correctly identified. So this is a clean, honest negative: an exogenous, orthogonal macro signal that simply does not time forward returns well enough — NOT a disguised price lane.
4. **Data honesty held:** the nowcast used ONLY point-in-time ETF prices, zero macro-database/revision surface, zero lookahead. A richer real-economic-series nowcast would need a paid PIT vendor (scope noted §3), but the leak-free proxy version was sufficient to reach a confident verdict.

**Cumulative finding for the slate:** the exogenous-macro input class, built honestly from tradeable proxies, ALSO tops out ~0.5 on this eval surface — same ceiling as the best price lanes. This strengthens the structural conclusion that the eval surface itself caps these archetypes near ~0.5–0.6, rather than any one lane being uniquely weak. The one lane that genuinely cleared higher (PEAD 0.926) remains the outlier worth pressing.
