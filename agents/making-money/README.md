# making-money agent

Agent workspace for income generation experiments and side-project automation.

## Purpose
Researches monetizable ideas, builds small experiments, runs outreach campaigns, and tracks results.

## Contents
- `*.py` — top-level business scripts (bizscript.py, biz2.py, merge_results.py, etc.)
- `live-copilot/` — live sales copilot tool (context engine, overlay, providers)
- `outreach/` — email/LinkedIn outreach scripts and batch data
- `build/` — experiment build dirs (exp1-pseo, exp2-loop, exp3-chrome)
- `research/` — research reports and analysis docs
