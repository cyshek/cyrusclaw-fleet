# Product Sheet — SyncDesk (demo context doc)

**What it is:** Two-way sync between help desks (Zendesk, Intercom, Freshdesk) and CRMs (Salesforce, HubSpot). No middleware, no Zapier.

## Pricing
- Starter: $12/user/mo — 1 help desk + 1 CRM, standard fields
- Growth: $24/user/mo — custom field mapping, two-way write-back, SLA alerts
- Enterprise: custom — SSO/SAML, audit logs, dedicated success manager

## Differentiators vs Zapier
- Native field-level mapping UI (no per-zap maintenance)
- Two-way write-back (Zapier is mostly one-direction without heavy setup)
- Sub-second sync vs Zapier's polling delays
- No per-task pricing that balloons at scale

## Security
- SOC 2 Type II
- Encrypted at rest (AES-256) + in transit (TLS 1.3)
- No training on customer data
- PII field-level redaction option
- Trust center: trust.syncdesk.example

## Typical ROI
- Removes ~2 manual handoffs per ticket
- Customers report payback in ~6 weeks for a 40-seat team
