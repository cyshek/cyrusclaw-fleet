# travel agent

Agent workspace for travel planning automation.

## Purpose
Researches trips, finds deals, builds itineraries, and monitors travel data.

## Contents
- `trips/` — trip planning docs and itineraries
- `scripts/` — utility scripts (e.g. YouTube research fetcher)
